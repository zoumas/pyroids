# Pyroids

An Asteroids clone built with Python and Pygame.

Following the [Build Asteroids using Python and Pygame](https://www.boot.dev/courses/build-asteroids-python) course on [boot.dev](https://www.boot.dev).

## Prerequisites

- [Python 3.13+](https://www.python.org/)
- [uv](https://docs.astral.sh/uv/)

## Getting Started

```sh
uv sync
uv run python main.py
```
